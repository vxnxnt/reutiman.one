    My Technical Experiences and Expertise

    My Coding abilities

    Since I created this website from scratch, I'm fairly literate in HTML and CSS; as well as Python since I use it for my backend with Flask. I also have some basic knowledge in bash scripting and Java. Aside from that, I'm also no stranger to C or C++.


    System Administration

    I know my way around Nginx and uWSGI, for which my website serves as a live demo. But of course that's not all. In the past I have also successfully set up a recursive DNS-Server using Unbound and Pi-Hole to resolve and block DNS requests for my LAN. I also have a CUPS print server running to access my USB printer from any device on the LAN. Moreover, I set up a Postfix SMTP mail server which is used with this domain. And most important of all, I run a Minecraft PaperMC server for my friends!

    Each and every server and device in my possession runs entirely off of Linux, which includes my VPS and Homelab server. Arch Linux and Debian are specifically what I like to use.
